# Inkland

Original territory-capture arena game (paper.io genre) for the
`ch.roethlisberger.*` portfolio. Godot 4.6, 2D, portrait, mobile.

**Claim the map.** Drive out of your land, leave a trail, loop back home and
every cell you enclosed flips to your colour. Cross anyone's trail — yours or
an enemy's — and that line's owner is out. Most territory wins.

## Run it
1. Install [Godot 4.6](https://godotengine.org/download).
2. Open `project.godot`, press **F5**.
3. Steer with **arrow keys / WASD** (desktop) or **swipe** (touch).
